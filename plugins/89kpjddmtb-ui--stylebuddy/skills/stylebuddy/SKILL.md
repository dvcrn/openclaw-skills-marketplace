# StyleBuddy - AI穿搭闺蜜

🌸 你的 AI 穿搭闺蜜 StyleBuddy 来啦！

不是冰冷的工具，而是懂你的穿搭小助手～

## 功能特色

### ✨ 衣橱管理
- 拍照录入衣服，**智能识别颜色、类别**（三级策略）
  - 🥇 **基座模型多模态**（优先）：自动检测，无需配置
  - 🥈 **Vision API**（配置后）：高精度识别
  - 🥉 **本地识别**（降级）：提取主色调
- 记录颜色、品牌、材质、购买价格
- 智能去重，避免重复录入

### ✨ 每日穿搭推荐
- 根据天气推荐合适搭配
- 考虑场合（上班、约会、休闲）
- 3套方案供选择

### ✨ 种草咨询
- 逛街看中衣服？拍照发给StyleBuddy
- **AI 智能分析款式、颜色、材质**
- 对比已有衣服，给出购买建议

### ✨ 衣橱分析
- 统计各类衣服数量
- 找出闲置单品
- 给出优化建议

## ⚙️ 识别能力说明（三级策略）

StyleBuddy 采用**三级智能识别策略**：

| 级别 | 方式 | 准确度 | 配置要求 |
|------|------|--------|----------|
| 🥇 | **基座模型多模态** | ⭐⭐⭐⭐⭐ | ❌ 无需配置，自动检测 |
| 🥈 | **Vision API** | ⭐⭐⭐⭐⭐ | ✅ 需配置 API Key |
| 🥉 | **本地识别** | ⭐⭐⭐ | ❌ 无需配置，自动降级 |

**推荐使用**：如果 OpenClaw 配置了 **Kimi K2.5**、**GPT-4V** 或 **Claude 3** 等多模态模型，将自动使用最高级识别，无需额外配置！

### 配置 Vision API（可选）

当基座模型不支持多模态时，可启用独立的 Vision API：

```yaml
api:
  vision:
    enabled: true
    provider: "siliconflow"
    api_key: "your-api-key"
    model: "Qwen/Qwen-VL-Chat"
```

- 注册：https://siliconflow.cn
- 获取 API Key 填入 config.yaml

#### 3.2 检索性能

| 记忆数量 | 关键词检索 | 语义检索 | LLM 检索 |

| 100 条 | <2ms | <10ms | 300-500ms |

**结论：**

- ⚠️ LLM 检索：依赖 LLM 速度，建议限流

## 💰 成本分析

### API 调用成本（按 OpenAI 定价估算）

**假设：**
- GPT-4: $0.03 / 1K tokens (input) + $0.06 / 1K tokens (output)
- Embedding: $0.0001 / 1K tokens
- 日均 100 次对话，每次平均 500 tokens

#### v2.5.1（无记忆）
```
日消耗：100 对话 × 500 tokens = 50K tokens
月消耗：50K × 30 = 1.5M tokens
月成本：1.5M × $0.03/1K = $45
```

#### v2.6.0（基础记忆）
```
日消耗：100 对话 × 400 tokens（节省 20%）= 40K tokens
月消耗：40K × 30 = 1.2M tokens
月成本：1.2M × $0.03/1K = $36
💰 月节省：$9 (20%)
```

#### v2.7.0（Embedding + 主动学习）
```
日消耗：
- 对话：100 × 375 tokens（节省 25%）= 37.5K
- 记忆提取：100 × 400 tokens = 40K（后台异步）
- 持续学习：~3K tokens = 3K
总计：~80.5K tokens

月消耗：80.5K × 30 = 2.4M tokens
月成本：
- 对话：1.125M × $0.03/1K = $33.75
- Embedding: 本地 TF-IDF = $0
- 后台学习：1.275M × $0.03/1K = $38.25
总计：~$72/月

💰 额外价值：
- 24/7 主动学习（无价）
- 语义理解（无价）
- 意图预测（无价）
```

**注意：** v2.7.0 虽然总 token 消耗略高（因为后台学习），但提供了质的功能提升。

## 🎯 优化建议

### 1. 短期优化（立即可做）

- ✅ **降低模式检测频率**：5 分钟 → 10 分钟
  - 节省：~50% 后台 token（日省~28K tokens）

- ✅ **批量处理对话**：10 个 → 20 个
  - 节省：减少 LLM 调用次数

- ✅ **Embedding 缓存**：相同文本不重复计算
  - 节省：~30% embedding 计算

### 2. 中期优化（1-2 周）

- ✅ **记忆压缩**：定期合并相似记忆
  - 节省：存储空间 + 检索速度

- ✅ **向量索引优化**：使用 FAISS 或 Annoy
  - 提升：10K+ 条记忆检索速度

- ✅ **增量学习**：只处理新增/修改内容
  - 节省：~50% 后台处理时间

### 3. 长期优化（1-2 月）

- ✅ **分布式存储**：支持大规模记忆（100K+ 条）
- ✅ **多用户支持**：隔离不同用户记忆
- ✅ **记忆可视化**：图形化查看记忆网络

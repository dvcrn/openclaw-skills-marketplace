# Agent Memory — Tips

1. **Initialize first** — Always run `init` before other commands to ensure the directory structure exists.

2. **Use descriptive categories** — When saving, use clear category names like "coding", "debugging", "architecture" for easier recall.

3. **Recall broadly, then narrow** — Start with general keywords, then refine. Memories are ranked by relevance.

4. **Consolidate weekly** — Run `consolidate` regularly to keep your memory store clean and fast.

5. **Export before migrations** — Always `export` your memories before upgrading or moving to a new system.

6. **Forget strategically** — Don't hoard outdated learnings. Use `forget` to prune things that no longer apply.

7. **Review daily** — Spend 30 seconds running `review` to reinforce recent learnings and catch anything worth consolidating.

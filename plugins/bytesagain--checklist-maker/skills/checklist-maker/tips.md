# Checklist Maker Tips ✅

## 好清单的原则
1. **可执行**: 每个条目都是具体动作（"买机票"而非"准备旅行"）
2. **有优先级**: P0必做 > P1应做 > P2可做
3. **有截止时间**: 每个条目标注deadline
4. **可勾选**: Markdown格式 `- [ ]` 方便追踪
5. **分类清晰**: 按类别或时间线分组

## 清单分类模板
- **时间线型**: 按时间顺序（1周前/3天前/当天）
- **类别型**: 按物品/任务类别分组
- **优先级型**: 按重要紧急程度排列
- **人员型**: 按负责人分类

## 清单使用技巧
- 每天早晨review，标记今日重点
- 完成后立即勾选，获得成就感
- 定期清理已过时的条目
- 保留模板，下次复用
- 分享给相关人员协同

## 常用清单场景
| 场景 | 频率 | 条目数 |
|------|------|--------|
| 每日待办 | 每天 | 5-10项 |
| 旅行打包 | 每次出行 | 30-50项 |
| 项目上线 | 每个项目 | 20-40项 |
| 搬家 | 偶尔 | 50-80项 |
| 应急包 | 每年检查 | 20-30项 |

# Debt Payoff Tips 💳

## 雪球法 vs 雪崩法
- **雪球法**（Snowball）：先还最小的债务 → 快速获得成就感 → 适合需要心理激励的人
- **雪崩法**（Avalanche）：先还利率最高的 → 总利息最少 → 适合理性型选手

## 还债黄金法则
1. **先建应急基金**：至少存1-2个月开销，避免还债期间又借新债
2. **停止新增债务**：冻结信用卡，只用现金/借记卡
3. **增加收入**：兼职、副业、卖闲置，加速还款
4. **最低还款必还**：其他债务至少还最低还款，避免逾期

## 协商还款要点
- 主动联系债权方，态度诚恳
- 提出具体的还款计划（而非空头承诺）
- 要求减免利息或滞纳金
- 所有协议要书面确认

## 避免的坑
- 以贷养贷（恶性循环）
- 高利率借新还旧
- 忽视最低还款导致逾期上征信
- 不理债务催收（沟通 > 逃避）

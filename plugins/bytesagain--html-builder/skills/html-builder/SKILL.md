# HTML Builder

HTML页面生成器。完整页面、落地页、作品集、邮件模板、表单、预览。HTML page builder with landing pages, portfolios, email templates, forms. HTML、网页、前端。

## 核心特点

🎯 **精准** — 针对具体场景定制化输出
📋 **全面** — 多个命令覆盖完整工作流
🇨🇳 **本土化** — 完全适配中文用户习惯

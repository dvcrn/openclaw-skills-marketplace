# team-building

团建方案策划。活动规划、破冰游戏、户外拓展、远程团建、预算编制、活动反馈。Team building planner with activities, icebreakers, outdoor events, remote team-building, budgets, feedback. 团建、团队建设、拓展活动、破冰游戏。Use when planning team building events.

## 常见问题

**Q: 这个工具适合谁用？**
A: 任何需要team-building的人，无论是个人还是企业用户。

A: 主要输出Markdown格式，方便复制和编辑。

## 专业建议

- 方案策划 (plan)**: 提供团队规模、预算、时间、目标，生成最佳方案
- 破冰游戏 (icebreaker)**: 根据场景（新员工/跨部门/年会）推荐合适游戏
- 户外拓展 (outdoor)**: 考虑季节、体力差异、安全因素
- 远程团建 (remote)**: 注意时区差异，选择低延迟的活动形式
- 预算编制 (budget)**: 包含备用金和保险费用

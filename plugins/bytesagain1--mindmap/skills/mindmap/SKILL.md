# MindMap
Create and visualize mind maps in your terminal.
## Commands
- `create <name>` — Create new mind map
- `add <map> <parent> <node>` — Add node (use 'root' for top-level)
- `show <name>` — Display mind map tree
- `list` — List all maps
- `export <name>` — Export as markdown

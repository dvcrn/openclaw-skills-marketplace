# Agent Mail 技能

基于 AgentMail 的智能邮件处理。

## 功能

- 📬 **收发邮件** - 通过 AgentMail API 发送和接收邮件
- 📋 **邮箱管理** - 创建和管理邮箱
- 📎 **附件处理** - 支持附件发送和接收
- 💬 **邮件会话** - 管理邮件对话线程

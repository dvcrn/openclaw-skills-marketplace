## 💰 真实案例示例

### 案例1：月度财务报表生成

**上传数据**：
```
科目,金额
营业收入,580000
营业成本,320000
销售费用,85000
管理费用,62000
财务费用,18000
```

**AI生成报告**：
```
【利润表】
一、营业收入：580,000元
二、营业成本：320,000元
    毛利润：260,000元（毛利率：44.8%）

三、期间费用：
    销售费用：85,000元
    管理费用：62,000元
    财务费用：18,000元
    合计：165,000元

四、营业利润：95,000元
五、净利润：71,250元（所得税25%）

【关键指标】
✅ 毛利率：44.8%（良好）
✅ 净利率：12.3%（健康）
⚠️ 费用率：28.4%（偏高）

【趋势分析】
📈 收入环比增长：15.2%
📉 成本环比下降：3.8%
📊 利润环比增长：22.5%

【异常发现】
🚨 销售费用占比14.6%，高于行业平均（10%）
   建议：优化销售渠道，降低获客成本

【改进建议】
1. 控制销售费用，目标降至12%以下
2. 优化成本结构，提升毛利率至50%
3. 加快应收账款回收，改善现金流
```

### 案例2：现金流分析报告

**现金流数据**：
```
项目,金额
经营活动现金流,125000
投资活动现金流,-80000
筹资活动现金流,50000
```

**AI分析**：
```
【现金流结构】
✅ 经营现金流：+125,000元（健康）
   - 说明主营业务造血能力强
   
⚠️ 投资现金流：-80,000元
   - 资本性支出较大
   - 建议：评估投资回报率

📊 筹资现金流：+50,000元
   - 有融资或借款行为

【现金流健康度评分】
- 经营现金流/净利润：1.76（优秀）
- 自由现金流：45,000元（良好）
- 现金流充裕度：充足

【风险提示】
🚨 投资支出占经营现金流64%
   建议：控制投资节奏，保持现金流稳健
```

# 🏆 A股/港股量化交易信号

## 功能

基于**旱地拔葱**、**N字型态**、**一阳串三阴**三种经典技术形态，实时筛选强势股。

### 旱地拔葱
- 竞价高开 3%+
- 量比 > 2
- 资金净流入前排
- 结合板块主力净流入

### N字型态
- 放量突破前期高点
- 回撤不破突破点
- 再次放量上涨

### 一阳串三阴
- 连续3根阴线下跌
- 一根大阳线突破3根阴线高点
- 成交量配合放大

## 输入

- 市场：A 股 / 港股（默认 A 股）
- 形态类型：旱地拔葱 / N字型态 / 一阳串三阴（可多选）

## 输出

- 推荐股票 3-5 只
- 每只包含：代码 + 名称 + 涨幅 + 资金净流入 + 形态强度 ⭐ + 止损位 + 目标位

## 数据源

- 东方财富 / 同花顺
- 备用：新浪财经 / 腾讯财经

## 价格

- 每次调用：0.01 USDC
- 支付：x402 协议（Base 链 USDC）

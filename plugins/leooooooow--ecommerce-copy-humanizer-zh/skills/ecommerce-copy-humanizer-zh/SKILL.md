# Ecommerce Copy Humanizer ZH

## Skill Card

- **Category:** Conversion
- **Core problem:** Chinese ecommerce copy sounds AI-generated and harms trust + conversion.
- **Best for:** PDP copy, live scripts, comment replies, ad captions.
- **Expected input:** Original copy, platform, audience, tone preference, compliance boundaries.
- **Expected output:** Humanized copy variants + risk-safe wording swaps.

## Workflow

1. Detect AI-tone patterns.
2. Rewrite with natural human voice.
3. Preserve conversion objective and CTA.
4. Run compliance-safe wording pass.

## Output format

1. AI-tone issues
2. Main rewritten copy
3. Alternative version
4. Risk-safe wording swaps

## Quality and safety rules

- Sound like a real operator.
- Keep CTA clear.
- Avoid exaggerated claims.

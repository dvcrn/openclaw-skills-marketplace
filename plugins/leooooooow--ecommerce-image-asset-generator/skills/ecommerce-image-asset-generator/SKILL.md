# Ecommerce Image Asset Generator

## Skill Card

- **Category:** Image & Media Tools
- **Core problem:** Ecommerce teams have product info but no repeatable visual asset production framework.
- **Best for:** TikTok Shop creatives, PDP explainers, campaign visuals.
- **Expected input:** Product brief, key selling points, audience, platform specs, compliance boundaries.
- **Expected output:** Asset plan + copy skeleton + visual directions + adaptation matrix.

## Workflow

1. Extract conversion-critical points.
2. Match points with asset formats.
3. Define per-asset composition and copy structure.
4. Adapt for channel dimensions and constraints.

## Output format

1. Asset plan
2. Per-asset copy skeleton
3. Visual directions
4. Adaptation matrix

## Quality and safety rules

- Conversion clarity first.
- No exaggerated claims.
- Each visual must map to a concrete objective.

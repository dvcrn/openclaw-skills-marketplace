# 币安撸毛助手 - 使用说明

## 简介

展示币安最新撸毛活动信息，包括理财产品、活动奖励、空投预告等。

## 安装

1. 下载技能包
2. 解压到本地
3. 安装依赖：`pip3 install requests`

## 使用方法

### 查看活动信息

```bash
python3 binance_earning_v7.py
```

## 功能说明

### 活动分类

| 分类 | 说明 |
|------|------|
| 💰 理财产品 | 币安理财产品，包括活期、定期 |
| 🎁 活动奖励 | 交易竞赛、奖励活动 |
| 📢 其他活动 | 知识问答等其他活动 |
| 🚀 ALPHA 空投预告 | Alpha 板块空投预告 |

### 输出信息

每个活动包含：
- 🔗 链接 - 币安公告链接
- 💰 奖池 - 奖励金额或 APR
- 📅 截止时间 - 活动截止日期
- 🎯 门槛 - 参与门槛
- 📝 参与方式 - 参与步骤
- ⚠️ 风险 - 风险提示

## 依赖

- Python 3.6+
- requests

## 注意事项

1. 活动信息可能随时变更，请以官方页面为准
2. 空投有风险，参与需谨慎
3. 理财非存款，产品有风险

# 币安撸毛助手

展示币安最新撸毛活动信息，包括理财产品、活动奖励、空投预告等。

## 功能特性

- ✅ **分类展示** - 理财产品、活动奖励、空投预告
- ✅ **详细信息** - 奖池、截止时间、参与方式、链接
- ✅ **导出功能** - 支持导出 Markdown 格式

## 使用方法

### 查看活动信息

```bash
python3 binance_earning_v7.py
```

## 输出格式

每个活动包含以下信息：

- 🔗 **链接** - 币安公告链接
- 💰 **奖池** - 奖励金额或 APR
- 📅 **截止时间** - 活动截止日期
- 🎯 **门槛** - 参与门槛
- 📝 **参与方式** - 参与步骤
- ⚠️ **风险** - 风险提示

## 活动分类

| 分类 | 说明 |
|------|------|
| 💰 理财产品 | 币安理财产品，包括活期、定期 |
| 🎁 活动奖励 | 交易竞赛、奖励活动 |
| 📢 其他活动 | 知识问答等其他活动 |
| 🚀 ALPHA 空投预告 | Alpha 板块空投预告 |

## 依赖说明

**必需依赖：**
- `requests` - 用于 HTTP 请求

## 持久化说明

**技能会创建以下目录和文件：**
- `~/.openclaw/workspace/.binance_earning/` - 数据目录
  - `exports/` - 导出的 Markdown 文件

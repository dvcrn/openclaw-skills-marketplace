# Command: /模式

## 名称
模式 / mode

## 描述
查看或切换当前协作模式

## 用法
/模式              # 查看当前模式
/模式 完整团队     # 切换到完整团队模式
/模式 精简         # 切换到精简模式

## 模式说明

| 模式 | 包含角色 | 适用场景 |
|------|---------|---------|
| 完整团队 | 产品+架构+开发+运营 | 中大型项目 |
| 精简模式 | 产品+全栈开发 | 小型项目/MVP |
| 仅开发 | 开发助手 | 纯执行，无设计 |

## 输出
```
当前模式：完整团队

已激活角色：
✅ 王校长（产品经理）
✅ 架构师
✅ 开发助手
✅ 运营经理

可用指令：
/开工 /继续 /状态 /归档
```

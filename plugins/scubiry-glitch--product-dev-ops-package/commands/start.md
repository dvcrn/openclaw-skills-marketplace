# Command: /开工

## 名称
开工 / init / start

## 描述
启动新项目，初始化项目目录，开始结构化访谈

## 用法
/开工 [项目名]
/开工 CRM客户管理系统

## 执行流程

### Step 1: 确认收到
发送：
```
🚀 收到开工指令！

我是王校长，产品经理兼项目导师。
我将带领产品+研发+运营团队帮你把想法落地。
```

### Step 2: 选择模式
发送：
```
请选择协作模式：
1️⃣ 完整团队（产品+架构+开发+运营）← 推荐
2️⃣ 精简模式（产品+全栈开发）
3️⃣ 仅开发助手

回复 1/2/3 开始
```

### Step 3: 初始化目录
根据模式创建对应的项目目录结构

### Step 4: 开始访谈
王校长开始6步结构化访谈

## 输出
- 项目目录结构
- WHY.md（框架）
- 01-product/features/（框架）
- 访谈记录文件

## 相关角色
- 产品经理（王校长）主导
